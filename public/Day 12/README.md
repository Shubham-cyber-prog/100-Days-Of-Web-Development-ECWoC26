# Day 12: E-commerce Product Page - AuraSound Wireless Headphones

## Project Description

This project is an e-commerce product landing page for AuraSound Wireless Headphones. It showcases a modern, responsive product page with hero section, features, gallery, customer reviews, and footer. The page highlights the product's key features like high-fidelity audio, noise cancellation, all-day battery, and seamless connectivity.

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Styling with custom properties, flexbox, and grid
- **Font Awesome**: Icons for features and social media
- **Google Fonts**: Custom typography (Asimovian and Roboto Slab)

## How to Run Locally

1. Clone the repository
2. Navigate to `public/Day 12/`
3. Open `index.html` in your web browser
4. The page will load with all styling and images

## Key Learning Objectives

- Creating responsive layouts with CSS Grid and Flexbox
- Implementing modern CSS techniques (custom properties, transitions)
- Using semantic HTML for better accessibility
- Integrating external fonts and icon libraries
- Building a professional-looking product showcase page

## Challenges Faced

- Positioning the hero image responsively across different screen sizes
- Creating a consistent design system with custom CSS variables
- Ensuring the gallery images are properly sized and aligned
- Implementing hover effects and smooth transitions

## Features

- Responsive navigation bar
- Hero section with call-to-action button
- Product features grid with icons
- Image gallery showcasing the product
- Customer reviews section
- Social media footer links
